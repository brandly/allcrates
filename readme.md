# allcrates

Generate a dependency graph for every Rust crate to be [viewed in `pm`](https://github.com/anvaka/pm).

## usage

```
$ git clone https://github.com/brandly/allcrates.git
$ cd allcrates
$ npm install

# produces .bin files in data/
$ node index.js
```
